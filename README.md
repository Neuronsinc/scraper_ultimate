# Scpraper Ultimate
#####  for DataScrap


------------


Apliación para renderizar base de datos de scraper y resumir reportería  en un apartado frontend y generar como archivo excel. 

##  instalación de paquetes -- backend
 `cd backend`
 `yarn install`
####  Iniciar backend
 `yarn start` 
 
## Instalación de paquetes -- frontend
 `cd frontend`
 `yarn install`
####  Iniciar frontend
 `yarn start`
